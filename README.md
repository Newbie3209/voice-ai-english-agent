# Voice AI English Agent

A fully offline voice-based conversational AI agent built using:

- Whisper (speech-to-text)
- Ollama + Llama3 (local LLM)
- pyttsx3 (text-to-speech)

## Features

- Speak through microphone
- AI responds with voice
- Runs locally without API cost
- Designed as a spoken English practice assistant

## Tech Stack

Python, Whisper, Ollama, Llama3, pyttsx3

## Run Locally

1. Install Ollama and download llama3
2. Create virtual environment
3. Install dependencies:

pip install -r requirements.txt

4. Run:

python voice_agent.py
